OFRECEMOS DIVERSOS PROYECTOS

Calculadora de Operaciones Básicas (Python)

Nivel de dificultad: 1 - Fácil

Crea una calculadora interactiva en Python que permita realizar operaciones básicas como suma, resta, multiplicación y división. Aprenderás estructuras de control y operaciones matemáticas.

Tecnologías: Python

Conversor de Unidades (Java)

Nivel de dificultad: 2 - Básico

Desarrolla un programa en Java que permita convertir unidades como metros a kilómetros o Celsius a Fahrenheit. Aprenderás control de flujo y estructuras condicionales.

Tecnologías: Java

Blog Simple con CRUD (PHP + MySQL)

Nivel de dificultad: 3 - Intermedio

Crea un blog donde los usuarios puedan crear, leer, actualizar y eliminar publicaciones usando PHP y MySQL. Aprenderás desarrollo web y operaciones CRUD.

Tecnologías: PHP, MySQL

Sistema de Gestión de Usuarios con Autenticación (Node.js + Express + MongoDB)

Nivel de dificultad: 4 - Avanzado

Crea un sistema de gestión de usuarios con autenticación utilizando Node.js y MongoDB. Aprenderás a crear APIs RESTful y a manejar autenticación con JWT.

Tecnologías: Node.js, MongoDB

Plataforma de Streaming de Video (Python + Flask + React + PostgreSQL)

Nivel de dificultad: 5 - Experto

Desarrolla una plataforma de streaming donde los usuarios puedan ver, cargar videos y dejar comentarios. Aprenderás desarrollo full-stack, React y PostgreSQL.

Tecnologías: Python, React, PostgreSQL

Red Social Completa (Node.js + React + MongoDB)

Nivel de dificultad: 5 - Experto

Crea una red social donde los usuarios puedan registrarse, hacer publicaciones, seguir a otros y comentar. Aprende a trabajar con Node.js, React y MongoDB para desarrollar una app completa.

Tecnologías: Node.js, React, MongoDB
